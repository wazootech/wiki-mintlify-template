## Before you begin

Install the [Wiki CLI](https://github.com/wazootech/wiki):

```bash
pip install wazootech-wiki
```

## Validate your wiki

```bash
wiki check --strict
wiki lint --strict
```

## Preview locally

```bash
npm run dev
```

Open `http://localhost:3000` to see your docs.

## Deploy to Mintlify

See [Deploy](#) — two options depending on whether you want full Mintlify features or a free static site on Vercel.
