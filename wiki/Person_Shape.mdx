# Person Shape

SHACL node shape validating `schema:Person` documents in this wiki. Each Person page must include:

- **givenName** — required string
- **familyName** — required string
- **title** — required string (maps to `schema:headline` in RDF)
- **description** — required string

Run `wiki check --strict` to validate pages against this shape.
