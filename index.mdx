# Wiki Docs Template

Edit semantic markdown in `wiki/`, validate with `wiki check`, publish with Mintlify or Vercel.

```bash
pip install wazootech-wiki
wiki check --strict
npm run dev
```

<Tip>
  Ready to make this your own? Start by editing this page. Update your `docs.json` to customize your site. Then fill out the [Quickstart](/quickstart) page.
</Tip>
